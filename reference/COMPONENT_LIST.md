# Component List — What You Need to Build

> Updated for v3.0 systems: deterministic combat, Tactics Cards, Player Boards, passive scouting, Reserve Track hidden march.

---

## Complete Prototype Component List

### Game Board
- [ ] 1x Large hex board (67 numbered hexes)
- [ ] Based on canonical Institute map (map.png reference)
- [ ] Terrain type clear at a glance (color coding by region)
- [ ] Special hex icons visible (watchtower, castle, cavern, river, discovery site)
- [ ] Red hex numbers (matching hexmap.PNG reference)
- [ ] Space for Proctor Card display and Round Tracker (1-12)

### Player Components (x4 sets for base game, one per House)
- [ ] 1x House Player Board per House
  - Passive ability, Special Unit, House Rule, Once Per Game power
  - Food tracking
  - Combat wins tally track
  - Objective tracking area (tally marks)
  - Unit count tracker (0-15)
- [ ] 13x Regular unit tokens (standing = Alert, laying = At Ease)
- [ ] 1x Flag-bearer token (distinct design, same standing/laying mechanic)
- [ ] 1x Special unit token (unique per House: Howler, Tactician, Archer, Shadow)
- [ ] 20x Control markers (colored discs)
- [ ] 1x Reserve Track board (3-space track)
- [ ] 1x Removal Marker (placed on hex when units enter Reserve Track)

### Resource Tokens
- [ ] 50x Food tokens (green)

### Cards
- [ ] 20x Tactics Cards (universal deck, distinct card back)
- [ ] 30x Proctor Cards (10 per Act, color-coded: Bronze / Silver / Gold)
- [ ] 16x Objective Cards (distinct card back, draw 2 keep 1)
- [ ] 12x Equipment Cards
- [ ] 5x Desperate Gambit reference cards (Scorched Earth, Martyrdom, False Flag, Last Stand, Scorpion's Tail)
- [ ] 4x Player Aid Cards (quick reference — turn structure, combat resolution, starvation table)

### Tokens and Markers
- [ ] 20x Discovery Tokens (face-down, placed on Discovery Sites at setup)
- [ ] 5x Valor Tokens
- [ ] 4x Castle Garrison tokens (face-down, strength 2-4, for neutral castles)
- [ ] 4x Trap Markers (for Scorpion's Tail gambit, hidden)
- [ ] 1x First Player Token
- [ ] 1x Round Tracker marker (moves along 1-12 track)
- [ ] Fortification markers (6+ recommended)
- [ ] Scorched Earth markers (2-3 recommended)
- [ ] Weakened/Hunger markers (for starvation tracking)

## Print-and-Play Prototype Shortcuts

For early playtesting before full production:

| Component | Shortcut |
|-----------|---------|

| Special Units | Unique color/shape per House (sticker label works) |

| Control Markers | Small colored discs or stickers |
| Reserve Track | Printed card or small board per player |
| Removal Markers | Colored ring or distinctive token per House |

## Card Specifications

### Tactics Cards (20)
- **Back:** Universal design (distinct from all other card types)
- **Front:** Card name, flavor text, effect, card type icon (Strength/Tactical/Terrain/Special)
- **Size:** Standard card (63x88mm)

### Proctor Cards (30)
- **Back:** Color-coded by Act (Bronze = Act I, Silver = Act II, Gold = Act III)
- **Front:** Card name, flavor text, effect, system interaction icons
- **Size:** Standard card

### Objective Cards (16)
- **Back:** Unique design (secret objectives — must not be identifiable from back)
- **Front:** Objective name, flavor text, condition, VP value, difficulty rating
- **Size:** Standard card

### Equipment Cards (12)
- **Back:** Equipment design (distinct from other decks)
- **Front:** Equipment name, flavor text, effect, hold/use indicator
- **Size:** Standard card

### Player Aid Cards (4)
- **Front:** Turn structure, order list, combat resolution steps, margin table
- **Back:** Starvation table by Act, resource conversion rates, Reserve Track range table, espionage actions
- **Size:** Large card or half-page reference

## Player Board Specifications

Each Player Board should include:

| Section | Content |
|---------|---------|
| **House Identity** | House name, sigil, thematic quote |
| **Passive Ability** | Always-on power (e.g., Mars: Red Rage) |
| **Special Unit** | Name and ability description |
| **House Rule** | Ongoing tactical benefit |
| **Once Per Game** | Powerful one-shot ability (with checkbox to track usage) |
| **Resource Tracks** | Food (0-12) |
| **Combat Wins** | Tally track (for 1 VP per win) |
| **Conversions** | Tally track (for 1 VP per conversion at game end) |
| **Objective Tracking** | Small area for tally marks (objective-specific tracking) |
| **Unit Summary** | Starting units, max 15, supply count |
